About Me

Halo, nama saya Fariz. Saya berusia 19 tahun dan berasal dari kota Gresik, Jawa Timur. Saat ini, saya sedang menempuh pendidikan di Universitas Pembangunan Nasional "Veteran" Jawa Timur, dengan jurusan Sistem Informasi di Fakultas Ilmu Komputer. Saya sangat tertarik dengan dunia teknologi, khususnya dalam pengembangan web. Oleh karena itu, saya memutuskan untuk membuat sebuah portofolio website untuk menunjukkan karya-karya saya dan memperkenalkan diri saya lebih lanjut di bidang ini. Di dalam website tersebut, Anda dapat menemukan informasi mengenai profil pribadi saya, latar belakang pendidikan, pengalaman kerja, serta keterampilan yang saya miliki.
